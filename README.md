# FounderFlow React MVP

FounderFlow is a beginner-first CRM concept for student founders and first-time entrepreneurs.

## What this demo includes

- A 3-question onboarding flow
- Beginner-friendly language for tracking opportunities
- A drag-and-drop opportunity board
- Follow-up reminders and coach-style prompts
- An AI-style message suggestion panel

## Stack

- React 18 rendered with native browser modules
- Component code in `src/App.js`
- Entry point in `src/main.js`

## Run it

Serve the folder with a simple local server so the browser can load the ES modules cleanly:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Notes

- This version does not require a local `npm` install in order to demo.
- React and supporting modules are loaded from CDN URLs at runtime.
